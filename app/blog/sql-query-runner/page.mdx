## A Common Use Case: Practicing a `JOIN`

Let's say you're learning how to join tables and want to find all orders placed by a specific customer.

1.  **Explore the Schema:** Use the schema explorer to see the `customers` and `orders` tables. Note that they can be joined on `customers.id` and `orders.customer_id`.
2.  **Write Your Query:** In the SQL editor, write your `JOIN` statement:
    ```sql
    SELECT
      c.name,
      o.order_date,
      o.total_amount
    FROM
      customers c
      JOIN orders o ON c.id = o.customer_id
    WHERE
      c.name = 'John Smith';
    ```
3.  **Execute and Analyze:** Click "Run Query". The results table will populate with the order date and total amount for all orders placed by John Smith. You can now experiment further by changing the customer name, adding more `JOIN`s, or using aggregate functions like `SUM()`.
