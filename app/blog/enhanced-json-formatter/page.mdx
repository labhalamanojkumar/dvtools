## Common Pitfalls and Solutions

### 1. Trailing Commas
**Problem**: JSON doesn't allow trailing commas
```json
{
  "name": "John",
  "age": 30,  // ‚Üê Invalid trailing comma
}
```

**Solution**: Our formatter automatically removes trailing commas and alerts you to the issue.

### 2. Single Quotes
**Problem**: JSON requires double quotes
```json
{
  'name': 'John',  // ‚Üê Invalid single quotes
  "age": 30
}
```

**Solution**: Automatic conversion to proper JSON format.

### 3. Comments in JSON
**Problem**: JSON doesn't support comments
```json
{
  "name": "John",  // This is invalid
  "age": 30
}
```

**Solution**: Detect and remove comments, or provide alternative formats like JSON5.

### 4. Large File Performance
**Problem**: Browser freezing with large JSON files

**Solutions**:
- Progressive loading and rendering
- Web Workers for background processing
- Virtual scrolling for large arrays
- Streaming for files over 100MB
