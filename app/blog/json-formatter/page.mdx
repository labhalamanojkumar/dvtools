
## Troubleshooting Common JSON Issues

### Common Errors and Solutions

**Error: "Unexpected token"**
```json
// Problem: Trailing comma
{
  "name": "John",
  "age": 30,
  "city": "NYC",  // ← This comma causes error
}
```

**Solution**: Remove trailing commas and ensure proper syntax.

**Error: "Invalid character"**
```json
// Problem: Single quotes instead of double quotes
{
  'name': 'John',  // ← JSON requires double quotes
  'age': 30
}
```

**Solution**: Use double quotes for all string values and keys.

**Error: "Circular reference"**
- Problem: Object references itself
- Solution: Use JSON replacer function to handle circular references
