# Student Participation Guidelines Image

## Current Files
- `student-participation-guidelines.svg` - Vector version (created)
- `student-participation-guidelines.jpg` - Raster version (needed for og:image)

## To Create JPG Version:

### Option 1: Using Online Converter
1. Open the SVG file in browser
2. Take screenshot or use SVG to JPG converter like:
   - cloudconvert.com
   - convertio.co
   - ezgif.com/svg-to-jpg

### Option 2: Using Design Tools
1. Open `student-participation-guidelines.svg` in:
   - Figma (figma.com)
   - Adobe Illustrator
   - Inkscape (free)
2. Export as JPG with dimensions: 1200x630px
3. Quality: 85-90%
4. Save as: `student-participation-guidelines.jpg`

### Option 3: Using Command Line (if imagemagick installed)
```bash
convert student-participation-guidelines.svg -resize 1200x630 -quality 90 student-participation-guidelines.jpg
```

## Image Specifications
- **Dimensions:** 1200x630px (optimal for social media sharing)
- **Format:** JPG for og:image, SVG as backup
- **File size:** Keep under 300KB for fast loading
- **Colors:** Gradient purple (#667eea to #764ba2)
- **Content:** Trophy, title, key points, prize badge

## Usage in News Article
The script references: `/news/student-participation-guidelines.jpg`
Both formats work, but JPG is preferred for social media previews.
